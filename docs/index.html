receipt-scanner

OCR a receipt and extract merchant, date and total — running entirely in your browser with tesseract.js. The original project (GitHub) is Python: OpenCV preprocessing + Tesseract + regex parsing. This page ports the parsing logic to JS so anyone can try it without installing anything.
